Acerca de Acerca de Nivel del agua

Este dispositivo mide el nivel de agua con un sensor ultrasónico y la unidad de medida es en centímetros (cm).

La tarjeta cambia de color y muestra un mensaje dependiendo del valor del dato medido:

"Lleno" para 32cm o más.

"Medio" para 16cm o más, pero menos de 32cm.

"Vacío" para menos de 16cm.

Contaminación del agua

Este dispositivo mide la contaminación del agua con un sensor de turbidez y la unidad de medida es en centímetros (cm).

La tarjeta cambia de color y muestra un mensaje dependiendo del valor del dato medido:

Color Rojo y "Sucio" para 32cm o más.

Color Amarillo y "Turbio" para 16cm o más, pero menos de 32cm.

Color Verde y "Limpio" para menos de 16cm.
